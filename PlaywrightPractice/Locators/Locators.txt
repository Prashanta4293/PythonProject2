NOTE:-
* Every web page create a DOM (Document Object Model) at run time.
* All types of locators directly (Xpath and CSS) or indirectly (Playwright built in locators) interact with DOM.

1- Playwright built in locators
   - get_by_alt_text()- Whenever we want to locate some image or logo in web page if having 'alt' attribute.
                  Syntax - page.get_by_alt_text("alt text")
     example- <img alt="company-branding">
   - get_by_text()- for inner text element which is in between anchor tag
                  Syntax - page.get_by_text("text")
     example- <p data-v-7b563373="" data-v-0af708be="" class="oxd-text oxd-text--p">Username : Admin</p>
                                                                                  -------------------
   - get_by_role()- this locator roles are predefined by W3C ARIA.
                  Syntax - page.get_by_role(role, name="visible text")

   - get_by_label()- locates form controls using the text of their associated label.
                  Syntax- page.get_by_label("Label Text")

   CSS Selectors/Locators (Important)
   ----------------------------------
   CSS- Cascading Style Sheets
   CSS- Works based on DOM
- tag id  ----   tag#id
- tag class  ----  tag.class
- tag attributes  ----  tag[attribute=value]
- tag class attributes  ----  tag.class[attribute=value]

NOTE-
- In all syntax 'tag' is optional.
- In CSS selectors/locators we can't use "" inside "", similarly we can't use '' inside ''.
- We can use "Selector hub" by installing it as a plugin from Chrome web store.
- For character start with syntax-
                                   tag[attribute^='some letters of the attribute value from start']
- For character end with syntax-
                                   tag[attribute$='some letters of the attribute value from end']
- For middle of string syntax-
                                   tag[attribute*='some letters of the attribute value from middle']
