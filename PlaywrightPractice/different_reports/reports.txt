Pytest-html reports
--------------------
Step 1: Install the plugin-
        command is "pip install pytest-html"
Step 2: Configure pytest.ini-
        Add "--html=myreport.html --self-contained-html --capture=tee-sys"
        [pytest]

addopts = --headed
          --browser=chromium
          --video=retain-on-failure
          --screenshot=only-on-failure
          --tracing=retain-on-failure
          --html=myreport.html --self-contained-html --capture=tee-sys
        Where --self-contained-html is use for to embeded all other external files inside html report itself
        and --capture=tee-sys is use for display the console results inside report
Step 3: Attach Screenshots on Test Failures (conftest.py)
Step 4: Create sample tests
Step 5: Run the Test and generate report
