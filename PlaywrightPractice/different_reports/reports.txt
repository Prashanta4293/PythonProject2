Pytest-html reports
--------------------
Step 1: Install the plugin-
        command is "pip install pytest-html"
Step 2: Configure pytest.ini-
        Add "--html=myreport.html --self-contained-html --capture=tee-sys"
        [pytest]

addopts = --headed
          --browser=chromium
          --video=retain-on-failure
          --screenshot=only-on-failure
          --tracing=retain-on-failure
          --html=myreport.html --self-contained-html --capture=tee-sys
        Where --self-contained-html is use for to embeded all other external files inside html report itself
        and --capture=tee-sys is use for display the console results inside report
Step 3: Attach Screenshots on Test Failures (conftest.py)
Step 4: Create sample tests
Step 5: Run the Test and generate report

Allure report
---------------
Step 1: Install Allure pytest plugin
        command is "pip install allure-Pytest"
Step 2: Install Allure command-line tool (to view and generate reports)
        - Go to "https://github.com/allure-framework/allure2/releases"
        - Download the latest version
        - Unzip the downloaded file
        - take the inside file which contains bin, config, lib, plugin
          to another place like C drive Program files
        - After that go to the bin folder and copy the path till bin folder
          because allure.bat file is there
        - then set the copied path in system variables 'Path'
Step 3: Configure Allure in pytest.ini
        add "[pytest]
             addopts = --alluredir=reports/allure-results"
Step 4: Create and Run your tests
Step 5: Generate and view the Allure report
        - Go to command prompt and give cmd "allure serve reports/allure-results" (temp report)
        - Permanent report cmd
          "allure generate reports/allure-results -o reports/allure-report --clean"
Step 6: Attach screenshots on test failures (conftest.py)
